1.Выведите результат последней выполненной команды;
    echo $?

2.Выведите 1 только если команда cmd выполнилась успешна;
    cmd && echo 1

3.Выведите 2 только если команда cmd выполнилась неуспешно;
    cmd || echo 2

4.Выведите 3 вне зависимости от результата предыдущей команды;
    cmd; echo 3

5.В файле записана строка с именем файла, создайте файл с именем из этого файла.
    Комментарий: file.txt содержит строку, которую буду использовать как имя файла.
    touch `cat file.txt`
