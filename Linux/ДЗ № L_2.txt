Вопросы по теме "Простые фильтры":
    1.Выведите из файла test первые десять уникальных строк.
        uniq test | head

    2.Выведите из файла test первые десять уникальных строк содержащих слово Hello.
        grep Hello test | uniq | head

    3.Выведите из файла test первые десять уникальных строк содержащих слово Hello и
    отсортируйте их.
        grep Hello test | uniq | head | sort

    4.В файл test постоянно дописываются строки, выведите его на экран так,
    чтобы постоянно выводились добавляемые строки содержащие слово Hello.
        tail -f test | grep Hello

    5.Файл test очень большой, выведите его так, чтобы можно было постранично просматривать.
        cat test | less

Вопросы по теме "Регулярные выражения":
    1. Сматчите строки состоящие из слов (слово состоит из букв или чисел), разделенных
    пробелом:
        - начинающиеся с abc
            egrep ^abc regular.txt

        - содержащие 3 цифры подряд
            egrep [0-9]{3} regular.txt

        - содержащие не менее 3 букв
            Комментарий: команда ищет строки содержащие слова с тремя буквами подряд
            egrep '[[:alpha:]]{3,}' regular.txt

        - содержащие test или testx
            egrep 'test|testx' regular.txt

        - не заканчивающиеся на символы a и z
            egrep '[^az]$' regular.txt

        - состоящие ровно из двух слов, где первое слово xyz или abc
            egrep '(^xyz|^abc) [[:alnum:]]$' regular.txt

        - в которых содержится слово test (помните про пробел и то что после последнего
        слова пробела нет) повторенное от 1 до 3 раз подряд
            Комментарий: подразумеваю, что строка начинается и заканчивается 'test'.
            egrep '^test( test){0,2}$' regular.txt

        - в которых есть слова test test, между которыми может быть число (любое),
        например test 999 test
            egrep '(^| )test [[:digit:]]* test( |$)]' regular.txt

        - начинающиеся и заканчивающиеся одной и более цифрами
            egrep '^[[:digit:]]+.*[[:digit:]]+$' regular.txt

        - начинающиеся со слова test, перед которым могут стоять цифры,
        например test или 99test
            egrep '^[[:digit:]]+test( |$)|^test( |$)' regular.txt
