Вопросы к теме "Файловая система":
    1.Создайте символическую ссылку в текущем каталоге на файл test, 
    находящийся на два каталога выше.
        ln -s ../../test ln-test

    2.Перенаправьте вывод команды echo test так чтобы оно не вывелось на экран
        echo test > test1

    3.Выведите занятое место папками в домашнем каталоге
        du -h ~

    4.Выведите занятое и свободное место на жестких дисках системы
        Комментарий: использую grep для фильтрации данных.
        df -h /dev/sda* | grep sd*

    5.Выведите список смонтированных файловых систем
        findmnt

Вопросы к теме "Поиск файлов":
    1.Выведите полный путь команды du
        which du

    2.Найдите все файлы с именем test в вашем домашнем каталоге и подкаталогах
        find ~/ -name test

Вопросы к теме "Работа с внешними дисками":
    1.Смонтируйте флешку sdb1 в каталог /mnt
        Комментарий: 1) <uid> - ввести id пользователя, либо имя пользователя.
                     <gid> - ввести id группы, либо имя группы.
                     2) подразумеваю, что директория mnt существует в домашней папке.
        sudo mount -o uid=<uid>,gid=<gid> /dev/sdb1 ~/mnt

    2.Отмонтируйте флешку из каталога /mnt
        sudo umount ~/mnt

    3.Вы не можете отмонтировть флешку из каталога /mnt поскольку кто-то читает файл 
    на флешке. Найдите эти процессы.
        lsof ~/mnt
